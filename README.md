### 配方参数详解

#### CN117044946A（凝胶微粒）

- 海藻酸钠:羧甲基纤维素钠 = **0.1-5:1**
- 交联剂（柠檬酸）:羧甲基纤维素钠 = **0.001-0.1:1**
- 粒度：100-1000μm 占比 **≥95%**
- 介质吸水倍率 **≥10倍**

#### CN119699584A（食用凝胶）

- W1 = m(海藻酸类)/(m(CMC)+m(海藻酸类))×100% → **12%-35%**
- f = m(Ca²+)/m(海藻酸类羧基) → **0.1-1.2**
- 对43%白酒吸附率 **≥700%**
- 原料浓度 **4%-15%**，粒径 **0.3-1mm**

#### CN119700647A（胃内驻留+转化）

- W1 = **13%-25%**
- W2 = m(益生菌)/总原料×100% → **≥5%（优选≥10%）**
- f = m(Ca²+)/m(海藻酸类羧基) → **0.2-1.5**
- 对43%酒精吸附率 **≥1000%**
- 凝胶弹性模量 **>1000Pa**（优选>2000Pa）

## 核心功效与优势

### 各专利技术亮点

#### CN117044946A - 凝胶微粒
- 胃内**稳定吸附酒精**
- 肠道内**结构不破坏、不二次释放酒精**
- 减少肝脏负担，全程锁酒排出

#### CN119699584A - 高吸附凝胶
- **高吸附率**，快速降低胃内游离酒精
- 耐胃蠕动研磨，**延长胃内停留**
- 延迟入血，减轻神经与肝脏损伤

#### CN119700647A - 驻留+转化双功能
- **吸附 + 转化双功能**
- 益生菌将酒精→乙酸→CO₂+H₂O
- 凝胶保护益生菌耐胃酸，实现**1+1>2**解酒效果
- 胃内驻留，延缓排空

### 与传统解酒产品对比

| 特性 | 传统产品（加速代谢） | 糖吉宴遇（物理吸附） |
|------|---------------------|---------------------|
| 作用原理 | 加速肝脏分解酒精 | 物理包裹酒精排出 |
| 乙醛产生 | 会产生乙醛中间产物 | 不产生乙醛 |
| 肝脏负担 | 增加肝脏代谢压力 | 无肝脏负担 |
| 安全性 | 可能有药物相互作用 | 物理作用，无副作用 |
| 适用人群 | 受基因限制（ALDH2） | 无基因限制 |
